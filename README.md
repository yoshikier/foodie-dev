# foodie-dev
shop developer by java

# start
```
git clone git@github.com:yoshikier/foodie-dev.git

cd foodie-dev
mvn clean install -DskipTests
cd foodie-api

```

##### run
###### mvn

```shell
mvn spring-boot:run
```

###### shell

```shell
sh deploy/run.sh
```
